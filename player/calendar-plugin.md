# Calendar Plugin

## Overview

**LostWilderness-Calendar** is the plugin that runs the **shared calendar** and keeps it in sync (e.g. as the "leader" when multiple servers use the same calendar). As a player you don't run Calendar by itself; you play on **Survival** or **Amplified**, which use the calendar. Calendar provides the date, seasons, and join-day tracking that those servers use.

---

## What you'll see

- On **Survival** and **Amplified**, all calendar-related behaviour (e.g. **/date**, **/season**, **/datejoined**, world events that run on calendar day) comes from this calendar.
- There are **no extra commands** that only the Calendar plugin adds; everything is exposed via Survival/Amplified (and Events). So from a player's point of view, the "Calendar" is just the fact that the server has one shared date and season system.

---

- Use the [Calendar and Time](calendar-and-time.md) guide for how **/date**, **/time**, **/season**, and the equator work. That's the player-facing side of what the Calendar plugin provides.
