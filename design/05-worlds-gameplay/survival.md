# Survival

- **Main overworld**; same **in-game chat** as Amplified (server-wide).
- **Calendar runs 24/7** (one server is leader, both read from MySQL).
- **No /home** — design is “crusade/horse-travel” feel; players use **lodestones** and travel rather than teleport. *“I'll just need to use lodestones bc I can never use f3 to go to a base.”*
- **Nether** and **End** are separate worlds (Paper); **build height** extended with datapacks (e.g. Nether/End down to y -64).
- **Difficulty** locked to **Hard**.
