# Amplified

- Reached **only** via **Crying Obsidian portal** from **Survival** (no /server from Lobby to skip).
- **“Discover a hidden realm”** custom **advancement** (replaces end gateway as the “second dimension” unlock).
- **Only ticks when someone is in it** (performance); when empty it doesn’t run the full tick loop.
- **Brotherhood** temple and **Bible library** are in Amplified (desert/mesa); world kept vague for “truth seekers.”
- **Hemisphere option:** Amplified can be **southern hemisphere** seasons so it feels distinct from Survival.
- **Only Bedrock SMP with Amplified** is a stated selling point.
- **Amplified Nether** and **Amplified End** use **Multiverse** and a **separate seed** (e.g. "stephenholbery"); **Multiverse Nether Portals** links Amplified Nether to Amplified overworld.
