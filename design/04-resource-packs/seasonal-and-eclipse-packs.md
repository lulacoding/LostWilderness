# Seasonal and eclipse packs

## Eclipse / Blood Moon

- Server **forces a red moon / dark sky pack** when the eclipse **starts**; when the event **ends** it sends an **empty pack** so the sky returns to normal.
- Once a player has **accepted** the eclipse pack, it can **auto-load on future eclipses** (Mojang loading screen can appear briefly on pack change).

## Seasonal (winter, autumn)

- **Winter:** Some **trees recoloured**; leaves **die in autumn**, **bare in winter**, **regrow in spring/summer** (plugin + pack).
- **Autumn:** Autumn leaves in the pack; **leaf litter block** considered for future.
- **Christmas:** **Chest sprites** Dec 24–Jan 1; **snow week** at Christmas or in winter = more rain (biome/plugin).

## Custom music

- Custom music as **.ogg** in the pack; **Bedrock compatible** if in Bedrock `sounds.json` format.

## Cosmetics (clan)

- **Undyed dog collar** → clan hex colour; **undye with wet sponge**; **Ghast harness** same idea (clan colour).

## Slabs

- **Vertical slabs** (and normal slabs) for every block type where it makes sense (QoL).
