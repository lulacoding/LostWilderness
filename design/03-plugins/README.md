# 3. Plugins & modules

All plugins and modules used or planned for Lost Wilderness: Calendar, portals, LWEvents (Eclipse, Fog, Frost, etc.), clans, lobby, anticheat, and others.

**Implementation status:** See [../../implementation-status.md](../../implementation-status.md) for what is implemented in code vs not (e.g. /eoc, /calender stats, /alliance, /war, /nomad, /clan list, entity passthrough).

---

- [Calendar](calendar.md) — Commands, MySQL sync, date format (1st January 1MC, Epoch), config, known bugs
- [Portals](portals.md) — Custom Portal API, Survival/Amplified plugins, MySQL tables, 1:1 coords, air pocket
- [LWEvents and Eclipse](lwevents-and-eclipse.md) — Eclipse/Blood Moon, Fog, Frost, Blizzard, Heatwave, Nether sickness, RoofWither
- [Clans and war](clans-and-war.md) — /clan, /alliance, /war, /nomad, Discord webhooks, no /home
- [Lobby and queue](lobby-and-queue.md) — VoidWorldGenerator, DeluxeMenus, queue plugin, PackServer
- [Anticheat](anticheat.md) — Vulcan, Negativity, BungeeGuard, Themis/Wither considered
- [Other plugins](other-plugins.md) — Multichat, Clearlagg, Multiverse, datapacks, Harbor removed
