# Lost Wilderness – Amplified Server (Player Guide)

## Overview

The **Amplified** server is the second world in the Lost Wilderness network. It shares the same **calendar**, **events**, **clans**, and **portals** as Survival. You use the **proxy** to connect and can travel between Survival and Amplified via **portals**. Commands here are similar to Survival except there is no **/resetcalendar** (calendar is controlled from the Survival/Calendar side).

---

## Commands (on Amplified)

| Command | Description |
|--------|-------------|
| **/date** | Current in-game date and days on server. |
| **/time** | Current in-game time. |
| **/datejoined** | When you joined and how many days you’ve been on the server. |
| **/season** | Current season at your location (and hemisphere). |
| **/lwhelp** | Help menu and status of Calendar, Events, Clans, Portals. |
| **/lwdebug** | Debug info (e.g. events). |
| **/lwplugins** | Plugin management (admin/op). |
| **/portals** | List your linked portals. |
| **/deleteportals** | Admin only. Delete all portals for a username. |
| **/clan** | Clan system – same as on Survival. See [Clans guide](player-guide-clans.md). |
| **/alliance** | Alliance commands (some may not be implemented). |
| **/linkrequest** | Internal; console/RCON only. |
| **/approve** | Approve a pending Discord link request. |
| **/deny** | Deny a pending Discord link request. |

---

## What you’ll experience

- Same **calendar**, **seasons**, and **equator** rules as Survival (Z position decides your season).
- Same **world events** (eclipse, storms, blizzard, fog, etc.) and sleep-blocking rules. See [Events guide](player-guide-events.md).
- **Clans** and **alliances** – same as Survival; your clan is shared across the network.
- **Portals** – you can build and use portals on Amplified to travel to Survival and back. See [Portals guide](player-guide-portals.md).
- **Discord linking** – **/approve** and **/deny** work the same as on Survival.

---

## Tips

- Connect through the **proxy** (e.g. port 25565) so portals and server switching work.
- Use **/lwhelp** to confirm that Calendar, Events, Clans, and Portals are available on this server.
